# 自己紹介 LP (Web3概論 第1回課題)

## 概要
本プロジェクトは、千葉工業大学の2026年前期「Web3概論」の第1回課題として作成した、自己紹介用のランディングページ（LP）です。

## 作成理由
授業の課題である「Antigravity + GitHub + Vercel のセットアップをして、自己紹介LPを作成・公開する」という要件を満たすために作成しました。
また、自分にとってAIとは何か、今後どのようにAIと向き合い、この授業を通して何を学んでいきたいかを言語化し、対外的に発信できるポートフォリオの最初の基盤とすることを目的としています。

## プログラムの説明
AIエージェント（Antigravity）と対話しながら、フレームワークを使用せずにVanilla（素の）HTML/CSS/JavaScriptのみを用いてシングルページウェブサイトとして構築しました。作成したコードはGitHub上でバージョン管理され、Vercelを介して全世界に公開できる仕組みになっています。

## ページの説明
本ページは、ダークモードを基調とし、サイバーでモダンなデザインで構成されています。スクロールに合わせたアニメーション（フェードイン等）を実装し、動的なユーザー体験を提供しています。

ページ内の主なセクションと記載している内容は以下の通りです：
- **About (自己紹介)**: 千葉工業大学所属。強み（順序立てて進めること）、興味関心（AI、特にLLM）について記載。
- **AI View (AIとはどのような存在か)**: 「割と物事に詳しい隣人・友人のような存在」としてのAI観を提示。
- **Future (AIとどう向き合いたいか)**: AIを活用しつつも頼りすぎず、自らの力を損なわない形で共に研究活動を行うというスタンスを記載。
- **Web3 (授業を通して探求したいこと)**: 人の役に立つAIアプリ開発や、AI・LLMによるコーディングへの意欲を提示。
- **Portfolio (成果物一覧)**: これまでに作成したプロジェクト（例: Web Design Evaluator Gemma）や今後の成果物をまとめるセクション。

## 構造
本プロジェクトは、静的なWebサイトとして以下のファイル構成となっています。

```text
my-intto/
├── index.html       # メインのHTMLファイル（ページの骨組みとテキストコンテンツ）
├── index.css        # スタイルシート（配色、レイアウト、アニメーション設定）
├── index.js         # JavaScriptファイル（スクロール時のフェードイン制御など）
├── avatar.png       # プロフィール用のアバター画像
├── game-title.png   # ポートフォリオ用のサムネイル画像（外部ゲームのタイトル画像）
└── README.md        # 本ファイル（プロジェクトの説明書）
```

## 使用技術・ツール
- **Frontend**: HTML5, CSS3, JavaScript
- **AI Assistant**: Antigravity (コーディング補助、ペアプログラミング)
- **Version Control**: Git, GitHub, GitHub CLI
- **Hosting / Deployment**: Vercel
